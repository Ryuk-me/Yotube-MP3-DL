# Yotube-MP3-DL
Youtube MP3 Downloader

# Install requirements
`$ pip install -r requirements.txt`

# Install ffmpeg 
 `Windows Users Run command prompt as admin and run the following code`
`$ choco install ffmpeg`

`MAC users`
`$ brew install ffmpeg`

`Linux users`
`$ sudo apt-get install ffmpeg`
